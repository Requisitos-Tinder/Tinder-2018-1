UC03 - Deletar Conta


Descrição: Este caso de uso permite que o usuário possa ser capaz de excluir sua conta.

Ator principal: Usuário e sistema.

Pré-Requisitos: O usuário deve possuir um perfil e precisa estar logado.

Fluxo Principal:
1. O usuário abre o aplicativo.
2. O usuário seleciona aba perfil.
3. O usuário seleciona a opção de configurações.
4. O usuário desliza a tela até a opção deletar conta.
5. O sistema exibe a tela de desativar, onde o usuário pode escolher desativar a conta ou congelar a conta.

Fluxo Alternativo 01: Deletar a conta
1. O usuário escolhe a opção de excluir minha conta.
2. O sistema exibe a tela de deletar conta.
3. O usuário escolhe o motivo da exclusão da conta.
4. O sistema mostra a mensagem de confirmação para excluir a conta.
5. O sistema deleta a conta.
6. O sistema exibe uma mensagem confirmando a exclusão da conta.

Fluxo Alternativo 02: Congelar Conta
1. O usuário escolhe a opção congelar a minha conta.
2. O sistema abre a tela principal.
3. O sistema exibe a mensagem ‘Seu cartão está oculto’.
4. O sistema exibe opção de ativar descoberta.

Fluxo de exceção: ---

Pós-condição: O perfil do usuário não existirá mais.
