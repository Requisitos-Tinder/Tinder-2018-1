UC02 - Login do Usuário


Descrição: Este caso de uso permite que o usuário possa se conectar ao sistema. Isso pode ser feito tanto pelo Facebook quanto pelo número de celular.

Ator principal: Usuário

Pré-Requisitos: O usuário não deve estar logado em uma conta.

Fluxo Principal: Login com Facebook:
1. O usuário acessa o aplicativo do Tinder.
2. O usuário seleciona a opção de “entrar com Facebook".
3. Os dados do Facebook são importados para o perfil do Tinder.

Fluxo Alternativo: Login com número de celular:
1. O usuário acessa o aplicativo do Tinder.
2. O usuário seleciona a opção de “entrar com número de celular”.
3. O usuário é redirecionado para a descoberta.


Regras de Negócio 01

Campo Formato Obrigatoriedade Valor
Número de celular Varia de acordo com os países. Sim O usuário não consegue acessar sua conta sem o número de celular.
Email exemplo@exemplo.com ou exemplo@exemplo.com.br Sim O e-mail deve ser válido e deve já ter sido cadastrado

Fluxo de exceção: Conta do Facebook removida:
1. O usuário acessa o aplicativo do Tinder.
2. O usuário seleciona a opção de “entrar com Facebook".
3. O conta do Facebook não existe mais.
4. O sistema volta para o passo 2 do fluxo principal.


Pós-condição: usuário logado no sistema.
