Configuração de Notificação


Título Configuração de Notificação
Objetivo Configurar as notificações que o usuário deseja receber.
Contexto Pré-condições:
1. O usuário deve possuir um perfil e estar logado.
2. Autorização no dispositivo para enviar as notificações.
Pós-condição:
1. O usuário receberá notificações quando algo acontecer na aplicação.
Atores 1. Usuário.
2. Sistema.
Recursos Acesso à Internet.
Episódios 1. O usuário acessa as configurações.
2. O usuário desliza a tela até a seção notificações.
3. O usuário escolhe se quer receber notificações quando tem novos matches.
4. O usuário escolhe se quer receber notificações quando recebe novas mensagens.
5. O usuário escolhe se quer receber notificações quando recebe curtidas nas mensagens.
6. O usuário escolhe se quer receber notificações quando recebe superlikes.
7. O usuário escolhe se quer que tenha vibração dentro do aplicativo.
8. O usuário escolhe se quer que tenha sons no aplicativo.
9. O usuário clica em ok para aplicar as modificações.
10. A tela volta para a aba perfil.
Restrições Fluxo intuitivo.
Exceção 1.A internet cai durante a ação.
2. O usuário fecha o aplicativo durante a ação.
