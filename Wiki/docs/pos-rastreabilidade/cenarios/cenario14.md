# Denunciar Perfil de Outros Usuários

<br />

|Título|Denunciar Perfil de Outros Usuários|
|------|------------------------------------|
|**Objetivo**|O usuário pode denunciar outros perfis.|
|**Contexto**|**Pré-condição:**<br />1. É preciso estar logado na aplicação <br />**Pós-condição:**<br />1. Após denunciar o perfil, o usuário não deverá visualizar mais o perfil denunciado.<br />2. O Tinder deve tomar providências de acordo com o tipo de denúncia.|
|**Atores**|Usuário.|
|**Recursos**|Acesso à Internet.|
|**Episódios**|1. O usuário abre o aplicativo.<br />2. O usuário irá visualizar o perfil a ser denunciado. <br />3. O usuário escolhe o motivo da denúncia: Mensagens inapropriadas, Fotos inapropriadas, Parece spam, Outra.<br />4. O tinder apresenta uma mensagem de reported.<br />5. O perfil é retirado da tela do usuário.|
|**Restrições**|Perfil do denunciado voltar a aparecer para o usuário que o denunciou.|
|**Exceção**|1. A internet parar de funcionar no meio da ação.<br />2. O usuário fechar o aplicativo antes de terminar a ação.|
