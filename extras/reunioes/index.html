Pautas de Reuniões

Reunião - 18/03


Data: 18/03

Horário de início: 20h30min.

Horário de término: 23h.

Pessoas presentes:

  • Amanda Muniz
  • Arthur Assis
  • Luciana Ribeiro
  • Vítor Cardoso
  • Waysman Silva

Pauta da reunião: elaboração do rich picture conceitual.

Discussões abordadas: a equipe imaginou o fluxo principal do tinder para usar como base na elaboração do rich picture conceitual.

Registro das decisões: O grupo escreveu o fluxo do tinder em ordem cronológica para decidir o que entraria no rich picture ou não. Utilizando a ideia de duas identidades e um verbo as conectando o seguinte fluxo foi feito:

  • Usuário - cadastra - TINDER
  • TINDER - cria - Perfil
  • Usuário - conecta - Redes Sociais
  • Usuário - gerencia - Perfil
  • Redes Sociais - se integram - TINDER
  • Usuário - avalia - Perfil (símbolo de positivo e negativo)
  • Usuário - é avaliado por seu - Perfil
  • Perfil Avaliado - gera - Match
  • Match - abre - Feed de Conversas
  • Conversas - existem entre - Usuários
  • TINDER - vende - Pacotes especiais
  • Usuário - compra - Pacotes especiais
  • Pacotes especiais - dão vantagens para - Usuários

Enquanto esse fluxo era criado surgiu a dúvida sobre colocar os concorrentes dentro do rich picture conceitual ou fazer um somente para isso. Essa dúvida acabou se tornando uma argumentação com as seguintes opiniões:

Discussão 01 - É necessário colocar concorrentes no Rich Picture?

P1 - É necessário colocar os concorrentes no Rich Picture para mostrar como o usuário escolhe qual aplicativo usar - Arthur

P2 - Não é necessário colocá-los, porque não se sabe se é possível encaixar os concorrentes no Rich Picture - Vítor

P3 - Não é necessário, porque no Rich Picture é necessário descrever o conceito da aplicação escolhida. Sendo assim, os concorrentes não se encaixam - Luciana

A conclusão dessa argumentação foi para não adicionar os concorrentes no rich picture conceitual.

Compromissos: reunião marcada para dia 19 de março às 20h30min para refatoração dos artefatos, levando em conta as críticas feitas pelos professores.

Reunião - 19/03


Data: 19 de março de 2018.

Horário de início: 20h30min.

Horário de término: 22h.

Pessoas presentes:

  • Amanda Muniz
  • Arthur Assis
  • Calebe Rios
  • Luciana Ribeiro
  • Vítor Cardoso
  • Waysman Silva

Pauta da reunião: divisão dos Rich Pictures.

Discussões abordadas: nesta reunião foi decidido quais rich pictures seriam feitos para o projeto final e quem seria responsável por elaborar cada um deles.

Registro das decisões: os rich pictures que serão entregues na próxima reunião são o de serviços pagos, concorrentes, visão social, cadastro, feed e mensagens, timeline do match, conceito e visão empresarial. A Amanda ficará responsável por fazer o rich picture de feed e mensagens e o da timeline do match. O Arthur ficará responsável pelo rich picture da visão empresarial do Tinder. Calebe ficará com o de serviços pagos. Luciana está responsável pelo conceitual e pela visão social. O Vítor ficará responsável pelo rich picture sobre os concorrentes. O Waysman ficará responsável pelo rich picture que mostra o fluxo de cadastro.

Compromissos: a entrega de cada um desses rich pictures será feita na próxima reunião que acontecerá amanhã às 20h30min.
